## 配置步骤

### 步骤 1: 配置环境变量 (5分钟)

#### 创建 .env 文件
在 `openclaw-self-optimization` 目录下创建 `.env` 文件：

```env
# GitHub 配置
GITHUB_TOKEN=your_github_token

# Telegram 配置 (可选)
TELEGRAM_BOT_TOKEN=your_telegram_bot_token
TELEGRAM_CHAT_ID=your_chat_id
```

#### 获取 GitHub Token
1. 访问: https://github.com/settings/tokens
2. 点击 "Generate new token"
3. 选择权限:
   - `repo` - 仓库访问
   - `workflow` - 工作流权限
4. 生成并复制 Token

### 步骤 2: 配置系统设置 (5分钟)

#### 编辑 config/settings.yml
```yaml
# GitHub 配置
github:
  username: "your-username"  # 你的 GitHub 用户名
  token: "your_github_token"  # GitHub Personal Access Token
  repository: "openclaw-self-optimization"  # 仓库名称

# 自动化配置
automation:
  analysis_schedule: "0 18 * * *"  # 每天 UTC 18:00
  recommendation_schedule: "0 18 * * 3,6"  # 每周三、六
  install_schedule: "0 18 * * 1,4"  # 每周一、四
```
