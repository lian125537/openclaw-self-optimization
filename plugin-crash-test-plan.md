## 📊 崩溃诊断工具

### 1. 崩溃捕获脚本
```bash
#!/bin/bash
# plugin-crash-test.sh
PLUGIN=$1
PORT=$2

echo "测试插件: $PLUGIN"
openclaw gateway --port $PORT > /tmp/plugin-$PLUGIN.log 2>&1 &
PID=$!

sleep 2
if kill -0 $PID 2>/dev/null; then
    echo "✅ 插件 $PLUGIN 正常"
    kill $PID
    return 0
else
    echo "❌ 插件 $PLUGIN 导致崩溃"
    tail -20 /tmp/plugin-$PLUGIN.log
    return 1
fi
```

### 2. 日志分析工具
```bash
# 分析崩溃日志中的模式
grep -i "error\|exception\|uncaught\|syntax" /tmp/plugin-*.log
```

### 3. 内存监控
```bash
# 监控插件内存使用
while true; do
    ps aux | grep "openclaw" | grep -v grep | awk '{print $4,$6/1024"MB"}'
    sleep 1
done
```
