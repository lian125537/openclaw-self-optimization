## 🎯 具体学习内容

### 1. Dify 核心功能
- **工作流引擎**: 可视化工作流编排
- **智能体开发**: 智能体创建和管理
- **模型集成**: 支持多种 LLM 模型
- **RAG 管道**: 检索增强生成

### 2. 架构设计
- **前端**: React + TypeScript
- **后端**: Python + FastAPI
- **数据库**: PostgreSQL
- **缓存**: Redis

### 3. 智能体开发
- **智能体定义**: 角色、技能、记忆
- **工作流编排**: 任务分解和执行
- **工具集成**: 外部 API 和服务
- **监控和调试**: 日志和指标
