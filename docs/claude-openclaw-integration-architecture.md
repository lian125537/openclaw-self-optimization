│  │  • 代码沙盒                         │   │
│  └─────────────────────────────────────┘   │

│  OpenClaw 核心                             │
│  • 会话管理                               │
│  • 工具执行                               │
│  • 消息路由                               │
└─────────────────────────────────────────────┘
```

### 集成点设计

#### 1. **钩子系统集成** (当前实现)
- **事件**: `message:preprocessed`, `gateway:startup`
- **功能**: 工具调用验证、错误处理、上下文监控
- **优点**: 无侵入式，完全兼容现有系统
- **限制**: 事件粒度较粗

#### 2. **插件系统集成** (推荐方案)
- **钩子**: `before_tool_call`, `before_agent_reply`, `on_error`
- **功能**: 细粒度控制，深度集成
- **优点**: 强大的控制能力，官方支持
- **状态**: 待实现

#### 3. **中间件服务** (备选方案)
- **协议**: WebSocket/REST API
- **功能**: 独立服务，完全解耦
- **优点**: 不影响主系统，易于维护
- **状态**: 备用方案

## 🔧 技术实现

### 当前实现: 钩子集成

```javascript
// 目录结构
workspace/
├── hooks/
│   └── claude-integration/
│       ├── HOOK.md          # 钩子元数据
│       └── handler.js       # 处理器实现
├── claude-core-porting/     # Claude组件
└── openclaw.json           # 配置
```

#### 配置示例
```json
{
  "hooks": {
    "internal": {
      "entries": {
        "claude-integration": {
          "enabled": true,
          "env": {
            "CLAUDE_CORE_PATH": "./claude-core-porting"
          }
        }
      }
    }
  }
}
```

### 推荐实现: 插件集成

```javascript
// 插件结构
claude-openclaw-plugin/
├── package.json
├── index.js                 # 插件入口
├── src/
│   ├── tool-validator.js    # 工具验证
│   ├── error-handler.js     # 错误处理
│   └── context-manager.js   # 上下文管理
└── hooks/
    └── claude-hooks.js      # 插件钩子
```
