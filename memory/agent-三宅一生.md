#### 1. 三宅一生品牌研究（夜墨整理）
- 品牌创立于1970年，创始人三宅一生（Issey Miyake）
- 四大产品线：Patchwork、B Pleats Please、A-POC、132 5.
- 设计哲学：服装是第三层皮肤、一块布理念、反时尚
- 面料创新：褶皱工艺、再生材料、蒸汽拉伸
- 品牌精神：舒适、自由、无束缚；"less is more"的日本诠释

#### 2. 服装设计理论体系（苏格拉底整理）
- 四要素：廓形（6种基本型）、面料（天然/合成/新型）、色彩（2026趋势色）、细节（领/袖/口袋/工艺）
- 廓形决定气质，H型=极简、A型=浪漫、X型=优雅、O型=休闲
- 2026趋势：Quiet Luxury静奢、Gorpcore户外、Digital Fashion数字、Bio-fabrication生物
- 服装设计史：从Chanel到Miyake到解构主义

#### 3. 时尚视觉Prompt库（宫崎骏整理）
- 四大摄影风格：Editorial杂志风、Catalog电商、Lookbook穿搭、Runway后台
- 镜头参数：85mm人像/35mm纪实/50mm标准/70-200长焦
- 光线词库：柔光箱/伦勃朗光/蝴蝶光/轮廓光/黄金时段
- 服装品类专属场景词库（女装/男装/童装）
- 实战Prompt模板：电商白底、杂志编辑、街拍、细节特写

#### 4. 服装消费者心理学（言心整理）
- 面料触感>视觉：消费者摸到面料才会产生信任感
- 首图决定跳失率：上身效果图 > 平铺图
- 面料图转化逻辑：看到面料细节 → 触发手感想象 → 下单
- 色彩情感：黑=安全、白=纯净、驼=品质、红=热情、蓝=信任
- 四类消费者：实用主义、形象追求、品质主义、环保意识
