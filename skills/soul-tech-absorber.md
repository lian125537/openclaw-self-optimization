# Soul Tech Absorber 技能

## 概述
用于吸收和分析 Soul APP 源码的技术架构和实现细节。

## 功能
1. **源码分析** - 分析 60K+ 反编译文件的结构和模式
2. **技术提取** - 提取 Soul APP 的核心技术和算法
3. **架构理解** - 理解 Soul APP 的系统架构设计
4. **代码学习** - 学习优秀的代码实现模式

## 使用方法
当需要分析 Soul APP 源码或提取技术时，自动启用此技能。

## 技术栈
- 文件系统遍历
- 代码解析
- 模式识别
- 架构分析

## 输出
- 技术报告
- 架构图
- 代码示例
- 最佳实践总结
