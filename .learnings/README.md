## 📊 系统概览

### 核心组件

1. **🚨 错误日志 (ERRORS.md)**
   - 自动捕获所有系统错误和用户纠正
   - 严重程度分级：🔴Critical 🟡Major 🟠Minor 🔵Optimization
   - 追踪修复状态：未修复 → 修复中 → 已修复

2. **📚 学习日志 (LEARNINGS.md)**
   - 记录技能掌握、认知洞察、优化创新
   - 重要性评分：1-10分
   - 验证状态：待验证 → 已验证

3. **📅 周度总结自动化**
   - 每周日自动分析
   - 独立代理执行，无需用户交互
   - 输出趋势分析和优化建议

### 🧰 工具套件

- `record_error.ps1` - 错误记录脚本
- `record_learning.ps1` - 学习记录脚本
- 每周cron任务自动分析

---

## 🎯 设计理念

> *"真正的创新是说不。" 但真正的改进是记录为什么说不，以及下次如何做得更好。*

### 为什么需要这个系统？

1. **记忆力脆弱**: AI助手每次会话都"重启" - 文件系统是我们唯一的连续记忆
2. **学习循环**: 错误 → 记录 → 分析 → 修复 → 验证 → 改进 → 再错误
3. **被动变主动**: 等待指令 vs 积极学习和优化自己

### 基于的先进模式

1. **WAL (Write-Ahead Log) 协议**: 在响应前记录关键信息
2. **Working Buffer**: 危险区上下文交换的完整记录
3. **proactive-agent**: Hal Stack中的主动学习架构

## 🔧 使用指南

### 手动记录错误

```powershell
.\.learnings\record_error.ps1 `
  -Level "Minor" `
  -Source "skill-name" `
  -Description "错误描述" `
  -Type "Error" `
  -Steps @("步骤1", "步骤2") `
  -Impact "影响描述" `
  -RootCause "根本原因" `
  -TempSolution "临时方案" `
  -PermSolution "永久方案"
```

### 手动记录学习

```powershell
.\.learnings\record_learning.ps1 `
  -Type "Cognitive Insights" `
  -Content "学到了一个重要模式" `
  -Source "用户交互" `
  -ApplicationScenarios @("场景1", "场景2") `
  -ExpectedBenefit "预期收益" `
  -ActionPlan @("计划1", "计划2") `
  -ImportanceScore 8 `
  -Verified
```

### 自动触发点

**应记录错误的情况**:
- 工具调用失败 (exec 返回非零码)
- 用户纠正:"不对，应该是..." 
- API调用限制/错误
- 权限问题

**应记录学习的情况**:
- 掌握新技能的高级用法
- 识别用户行为模式
- 发现效率提升的捷径
- 安全漏洞修复

## ⚙️ 技术架构

### 文件结构
```
.learnings/
├── ERRORS.md          # 错误日志
├── LEARNINGS.md       # 学习日志  
├── WEEKLY_SUMMARY.md  # 周度总结（自动生成）

├── record_learning.ps1# 学习记录脚本
└── README.md          # 本文件
```

### Cron配置
```json
{
  "name": "Self-Improvement Weekly Summary",
  "schedule": "0 0 * * 0",  // 每周日 00:00
  "sessionTarget": "isolated",
  "payload": {"kind": "agentTurn", "message": "执行周度总结..."}
}
```

### 与AGENTS.md集成
系统配置已添加到AGENTS.md的"Self-Evolution System"部分，确保每次会话都能看到系统状态。

## 🔍 质量保证

### 验证机制
1. **错误修复验证**: 每个修复必须有明确的验证步骤
2. **学习验证**: 重要性≥7的学习必须标记验证状态
3. **周度总结验证**: 检查趋势是否与实际体验一致

### 安全考虑
- 不记录敏感信息（密码、密钥、个人隐私）
- 所有文件存储在用户工作空间内
- 没有外部数据同步（除非用户配置）
